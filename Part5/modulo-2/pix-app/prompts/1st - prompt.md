# ROLE

Act as a Senior Front-end Engineer specializing in Angular 21.

# OBJECTIVE

Execute the tasks.

1. **Cleanup:** Replace the content of `app.component.html` generated by the CLI.
2. **Base Layout:** Create a simple semantic structure (`<nav>`, `<aside>`, `<main>`). Add a side menu with a single "Pix" item. Place the `<router-outlet>` alongside it.
3. **Routing:** Create a Standalone Component named `PixTransferComponent`. Configure `app.routes.ts` for the `/pix` route.
4. **Form:** In `PixTransferComponent`, create an HTML form using Signals to manage state, with the following fields: Pix Key, Amount, Scheduled Date, and a Confirm button.
5. **Styling Restriction (Critical):** Do not generate any CSS classes or inline styles. The HTML must be 100% pure and raw.
