# PAPEL
Atue como um Engenheiro Front-end Sênior Especialista em Angular 21.

# OBJETIVO
Execute as tarefas.


1. **Limpeza:** Substitua o conteudo do app.component.html gerado pelo CLI
2. **Layout Base:** Crie uma estrutura semantica simples (<nav>, <aside>, <main>). Adicione um menu lateral com um unico item "Pix". Coloque o <router-outlet> a lado
3. **Roteamento:** Crie um Standalone Component chamado PixTransferComponent. Configure o app.routes.ts para a rota '/pix'.
4. **Formulario:** No PixtransferComponent, crie um formulario HTML usando Signals para gerenciar estado, com os campos: Chave Pix, Valor, Data de Agendamento e Botao Confirmar.
5. **Restricao de Estilo (Critico):** Nao gere nenhuma classe CSS ou estilo inline. O HTML deve ser 100% puro e cru.
